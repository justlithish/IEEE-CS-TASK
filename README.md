Companion : AI Smartwatch Landing Page

A simple frontend landing page inspired by the **Companion AI Smartwatch** concept.
This project was built as a **learning exercise** to practice HTML, CSS, and JavaScript while creating a modern-looking product website.

The page presents an AI-enabled smartwatch idea that focuses on health, assistance, and everyday support through a clean and interactive UI.

Project Overview

* Concept-based AI smartwatch landing page
* Focus on visual storytelling and smooth scrolling
* Designed intentionally as a **learner-level project**
* Frontend only (no backend or real AI)


Technologies Used

* HTML5 – Structure
* CSS3 – Styling and animations
* JavaScript (Vanilla JS) – Interactivity
* jQuery – Basic DOM handling
* Lottie-Web – Vector animations


Features

* Preloader animation
* Scroll-triggered section animations
* Animated text and counters
* Interactive cards and hover effects
* Clean and readable layout
  

Project Setup

1. Clone or download the repository
2. Place all assets inside the `assets/` folder
3. Open `index.html` in any modern browser
4. (Optional) Run using a local server for better media support
   

Limitations

* No backend integration
* No real AI functionality
* Optimized mainly for desktop screens
* Local file paths need adjustment for deployment
  

Learning Outcomes

* Improved understanding of HTML structure
* Practiced CSS animations and layouts
* Used JavaScript for scroll-based interactions
* Built a real-world style landing page
